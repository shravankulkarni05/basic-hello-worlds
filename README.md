# basic-hello-worlds
Hello world app for various libraries/frameworks.

Things to know before you start :)

1. only simple hello world apps.
2. app must say "Hello world" on main screen.
3. should include only main component.
4. no routing, no styling, no extra things!
5. must work by running npm install and ng serve (or equivalent).
6. add build steps.
7. verify that the app once built can be served.
